# Orpheus Theme: Terminal

A retro terminal/hacker-style theme for [Orpheus](https://github.com/collectif-pixel/orpheus).

## Preview

- Green-on-black monospace aesthetic
- Fira Code font
- Fake terminal prompt with blinking cursor
- Album art rendered with grayscale + pixelated filter
- Scanline overlay
- Status bar with play/pause state and random PID

## Installation

```bash
orpheus add @littlemorphee/orpheus-theme-terminal
orpheus use @littlemorphee/orpheus-theme-terminal
```
